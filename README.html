1. Purpose

This program plays MIDI files using OPL3 emulation.

3. Downloading

The official home page of adlmidi is at http://iki.fi/bisqwit/source/adlmidi.html.
Check there for new versions.

Additionally, the most recent source code (bleeding edge) for adlmidi can also be downloaded by cloning the Git repository by:

  • git clone git://bisqwit.iki.fi/adlmidi.git
  • git checkout origin/release -b release
  • git checkout origin/master -b master
