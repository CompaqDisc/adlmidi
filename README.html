1. Purpose

AdlMIDI is a commandline program that plays MIDI files using software OPL3 emulation.

2. Key features

  • OPL3 emulation with four-operator mode support
  • FM patches from a number of known PC games, copied from files typical to AIL / Miles Sound System / DMX / Human Machine Interfaces / Creative IBK.
  • Stereo sound
  • Reverb filter based on code from SoX, based on code from Freeverb. A copy of either project is not needed.
  • Number of simulated soundcards can be specified as 1-100 (maximum channels 1800!)
  • xterm-256color support
  • Pan (binary panning, i.e. left/right side on/off)
  • Pitch-bender with adjustable range
  • Vibrato that responds to RPN/NRPN parameters
  • Sustain enable/disable
  • MIDI and RMI file support
  • loopStart / loopEnd tag support (Final Fantasy VII)
This project is developed in C++, but a GW-BASIC version is also provided (MIDIPLAY.BAS). This player was first implemented in GW-BASIC for a Youtube demonstration video! With alterations that take 15 seconds to implement, it can be run in QBasic and QuickBASIC too.

4. Downloading

The official home page of adlmidi is at http://iki.fi/bisqwit/source/adlmidi.html.
Check there for new versions.

Additionally, the most recent source code (bleeding edge) for adlmidi can also be downloaded by cloning the Git repository by:

  • git clone git://bisqwit.iki.fi/adlmidi.git
  • git checkout origin/release -b release
  • git checkout origin/master -b master
